# ✅ Doit — My To-Do List App

A clean, colourful daily to-do list web app built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just one file, ready to open in any browser.

---

## 🌟 Features

- **Welcome screen** with an animated front page to greet you each day
- **Add tasks** with a simple input field
- **Priority system** — mark each task as High, Medium, or Low priority
  - 🔴 **Red** — High priority
  - 🟢 **Green** — Medium priority
  - 🔵 **Blue** — Low priority
- **Numbered task list** so you always know how many tasks you have
- **Check off tasks** with a single click — completed tasks get a strikethrough
- **Delete tasks** when you no longer need them
- **Live progress bar** showing how many tasks you've completed
- **Today's date** displayed automatically on the task page
- Fully **responsive** — works on mobile and desktop

---

## 🚀 Getting Started

### Option 1 — Open locally

Just download `todo-list.html` and double-click it. It opens straight in your browser — no setup needed.

### Option 2 — Host on GitHub Pages

1. Upload `todo-list.html` to your GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Rename the file to `index.html` for it to load at your root URL
5. Your app will be live at:
   ```
   https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/
   ```

## 📝 How to Use

1. Open the app and click **"Click here to get started"**
2. Type your activity in the input box
3. Choose a priority level from the dropdown
4. Click **Add task** (or press Enter)
5. Click the **circle button** on the right to mark a task as done ✓
6. Click **✕** to delete a task
